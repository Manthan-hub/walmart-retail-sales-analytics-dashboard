# Retail Sales Insights: Walmart Dashboard

An **interactive Tableau dashboard** analyzing historical Walmart sales data from **February 2010 to November 2012**. 

![Dashboard](folder/dashboard-img.png)

[**View Interactive Dashboard**](https://public.tableau.com/views/RetailSalesInsightsWalmartDashboard/Dashboard?:language=en-US&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link)

Click the link above to open the interactive Tableau dashboard. Use filters to explore data by year. Hover over data points for detailed metrics.

## Key Features
- **KPI Cards**: Total Sales, Average Weekly Sales, and Max Weekly Sales
- **Monthly Sales Trend**: Year-over-year comparison revealing seasonal patterns
- **Impact of CPI on Sales**: Scatter plot analysis showing CPI’s relationship to sales
- **Total Sales by Year**: Comparison across 2010, 2011, and 2012
